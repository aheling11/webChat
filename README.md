# webChat
A web-based instant messaging application

## 技术栈
- spring boot
- netty
- mybatis

## 已完成

- 登录接口 /user/login
- 注册接口 /user/register
- 查找用户 /user/search
- 发送加好友请求 /user/addFriendRequest
- 用户查询收到的好友请求 /user/queryFriendRequest
- 用户处理收到的好友请求 /user/operFriendRequest
- 返回用户的好友列表 /user/queryFriends

## 待完成
- 聊天模块
